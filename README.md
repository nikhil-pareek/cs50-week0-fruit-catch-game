# 🍎 Fruit Catch Game (CS50 Week 0)

## 📌 Overview

Fruit Catch Game is my first programming project created using Scratch as part of Harvard's CS50 Week 0.

The objective is simple: move the bowl to catch falling fruits while avoiding losing lives. The game ends when all lives are exhausted.

---

## 🎮 Features

* Score tracking
* Lives system
* Random fruit spawning
* Game Over screen
* Custom blocks for code reuse
* Event broadcasting
* Sound effects

---

## 🧠 Concepts Learned

* Variables
* Loops
* Conditional statements
* Event-driven programming
* Custom blocks (abstraction)
* Problem solving
* Debugging

---

## 🛠️ Built With

* Scratch 3.0
